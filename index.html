Grive

An Open Source Linux Client for Google Drive

Download this project as a .zip file Download this project as a tar.gz file

This is the home page of Grive: an open source Linux client for Google Drive.

The purpose of this project is to provide an independent implementation of Google Drive client. It uses the Google Document List API to talk to the servers in Google. The code is written in standard C++.

Currently as of version 0.0.2, grive can only do one-side synchronization from the Google Drive to local directory. It can only download everything you have in your Google Drive right now.

Download

Current release is version 0.0.2. Fedora x64 binaries can be found here.

Usage

Grive requires the following libraries to run:

  • libstdc++
  • libcurl
  • json-c
  • openssl

For the first time running grive, you should use the "-a" argument to grant permission to grive to access to your Google Drive.

-----------------------
Please input the authenication code here

Go to the page. You will need to login to your google account if you haven't done so. After granting the permission to grive, the browser will show you an authenication code. Copy-and-paste that to the standard input of grive.

If everything works fine, grive will create a .grive file in your home directory. It will also start downloading files from your Google Drive to your current directory.
